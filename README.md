# Smarty extension for the Zed Editor

My attempt to port the [VS Code extension](https://github.com/aswinkumar863/smarty-vscode-support) for the Smarty templating language to Zed with Claude Code.

## Feature parity compared to the VS Code extension:
![Feature parity chart](https://i.imgur.com/R4El6fX.png "Feature parity chart")
## How to install:

1. Clone repository
2. Open the extension panel in Zed
3. Click the button in the upper right corner: 'Install Dev Extension'
4. Select the folder 'smart-zed-support'
5. Check if .tpl files have correctly selected 'Smarty' in the bottom right corner.


> Caution: I have no experience with developing extensions > for Zed. Use at your own risk!
